# LumenusErp

**LumenusErp** — это ERP-система для автоматизации малого бизнеса (на примере чайной), разработанная на **.NET 9** с использованием **Blazor** и **Entity Framework**. Проект обеспечивает унифицированное управление бизнес-процессами и интеграцию с **Telegram** для создания записей и бронирований.

## 🚀 Основные возможности

- **Интеграция с Telegram** – создание записей и бронирований через мессенджер.
- **Управление бизнесом** – персонал, финансы, склад, продажи, аналитика, документооборот.
- **Быстрое внедрение** – настройка системы за 3–7 дней.
- **Мобильный доступ** – работа с любого устройства.
- **Безопасность** – шифрование и защита данных.
- **Высокая доступность** – 99.9% аптайма и 24/7 поддержка.
